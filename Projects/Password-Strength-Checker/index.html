Tips for a stronger Password

  • Avoid using personal info like your name, birthday, or your pet's name
  • Avoid swapping a's with @'s and l's with 1's
  • Don't reuse passwords across different websites
  • Use a passphrase, if one is available, or string random words together like "apple couch shoe bottle asphalt"
  • Using a password manager is regarded as one of the safest way to store passwords online like BitWarden
  • Never enter real passwords into online tools, this checker runs 100% in your browser with zero server communication. Verify by disconnecting your Wi-Fi and testing it
  • Change passwords immediately if you are aware of a data breach as you may be vulnerable
